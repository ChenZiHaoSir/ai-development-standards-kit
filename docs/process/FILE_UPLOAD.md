# 文件上传/下载规范

本文件定义文件上传、下载的限制、校验、安全要求和用户体验。

## 上传限制

### 文件大小限制

| 场景 | 单文件限制 | 总大小限制 | 说明 |
| --- | --- | --- | --- |
| 头像 | 2MB | - | 头像图片 |
| 图片上传 | 10MB | 50MB | 文章配图、图片素材 |
| 文档上传 | 50MB | 200MB | 文档、报告 |
| 数据导入 | 100MB | 500MB | Excel、CSV、JSON |
| 视频上传 | 500MB | - | 视频素材 |
| 附件 | 50MB | 200MB | 一般附件 |

### 文件格式限制

| 场景 | 允许格式 | 禁止格式 |
| --- | --- | --- |
| 头像 | jpg, png, webp | gif, svg, bmp |
| 图片上传 | jpg, png, webp, gif | svg, bmp, ico |
| 文档上传 | pdf, doc, docx, xls, xlsx, ppt, pptx | exe, bat, cmd, sh |
| 数据导入 | csv, xlsx, json | 可执行文件 |
| 视频 | mp4, mov, avi, webm | - |

### 禁止上传的文件类型

绝对禁止上传以下文件类型：

- 可执行文件：`.exe`、`.bat`、`.cmd`、`.sh`、`.ps1`、`.msi`
- 脚本文件：`.js`、`.php`、`.py`、`.rb`、`.vbs`
- 压缩包含可执行内容：`.zip` 内含 `.exe` 等

## 上传校验

### 客户端校验

- 文件大小校验：超过限制时立即提示，不发送请求
- 文件格式校验：基于扩展名和 MIME Type 双重校验
- 图片预览：上传前显示缩略图，确认后上传
- 上传进度：显示进度条，显示已上传/总大小

### 服务端校验

- 文件格式校验：检查文件魔数（Magic Bytes），不依赖扩展名
- 文件大小校验：限制请求体大小
- 文件内容安全：使用杀毒扫描服务（如 ClamAV）扫描上传文件
- 文件名净化：去除特殊字符、路径分隔符，防止路径遍历

## 上传流程

```
用户选择文件
    │
    ▼
客户端校验（大小、格式）
    │  不通过 → 提示用户修改
    ▼
预览/确认
    │
    ▼
开始上传（显示进度条）
    │  超时/失败 → 提示重试
    ▼
服务端校验（格式、安全扫描）
    │  不通过 → 提示用户
    ▼
文件处理（压缩、缩略图、转码）
    │
    ▼
返回文件信息（URL、ID、元数据）
    │
    ▼
用户确认/继续操作
```

## 上传体验

- 支持拖拽上传（Drag & Drop）
- 支持批量上传（多文件选择）
- 上传中显示进度条和速度
- 上传失败显示具体原因，提供重试按钮
- 上传成功后显示文件预览或缩略图
- 支持取消上传（AbortController）
- 大文件支持分片上传（>50MB 自动分片）
- 断网恢复后自动重试未完成的上传

## 下载规范

- 文件下载必须有明确的下载按钮或链接
- 大文件下载必须显示进度
- 下载失败提供重试按钮
- 文件名使用原始文件名，禁止暴露服务器路径
- 下载接口必须校验权限（不能通过猜 URL 下载未授权文件）

## 安全约束

- 所有上传文件必须经过病毒扫描
- 上传文件必须存储在独立的文件服务/对象存储，不放在应用服务器
- 文件 URL 必须有时效性（签名 URL 或临时访问链接）
- 禁止通过上传文件执行脚本（服务器配置禁止上传目录执行代码）
- 上传文件必须有访问控制，不能公开访问未授权文件
- 文件元数据必须脱敏（IP、设备信息、原始路径）

## 禁止行为

- 禁止只做客户端校验不做服务端校验
- 禁止使用文件扩展名判断文件类型（必须检查魔数）
- 禁止上传文件存储在应用根目录或数据库
- 禁止文件下载接口无权限校验
- 禁止上传文件后不显示任何反馈
- 禁止无大小限制的上传（必须设置上限）
- 禁止大文件一次性上传不分片（>50MB 必须分片）
