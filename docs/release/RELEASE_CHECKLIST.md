# Release Checklist

## 发布前

- [ ] 版本号已更新
- [ ] CHANGELOG 已更新
- [ ] release notes 已准备
- [ ] 主分支干净
- [ ] CI 全部通过
- [ ] 依赖漏洞扫描通过
- [ ] 密钥扫描通过
- [ ] 数据库迁移已验证
- [ ] 回滚方案已确认
- [ ] 告警监控已确认

## 验证

- [ ] 核心路径验证通过
- [ ] 权限验证通过
- [ ] 日志脱敏验证通过
- [ ] 性能指标无明显回退
- [ ] 灰度策略已确认

## 发布后

- [ ] 灰度观察完成
- [ ] 错误率、延迟和核心业务指标正常
- [ ] 未完成事项已记录
- [ ] 发布问题已复盘并转化为规范或自动化检查
